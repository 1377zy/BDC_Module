Import Instructions
How to Import Leads
  1. Prepare your CSV or Excel file with lead data
  2. Select the file using the upload button
  3. Choose the lead source and type
  4. Configure duplicate handling settings
  5. Optionally map columns if your headers don't match our system
  6. Click "Import Leads" to begin the import process
Template Format

For best results, your file should contain the following columns:

  • First Name
  • Last Name
  • Email
  • Phone
  • Address (optional)
  • City (optional)
  • State (optional)
  • Zip (optional)
  • Vehicle Interest (optional)
  • Status (optional)
  • Notes (optional)

Common Issues
Troubleshooting
  • File Format: Ensure your file is saved as CSV or XLSX
  • Required Fields: First name, last name, and either email or phone are required
  • Special Characters: Avoid special characters in your data
  • Large Files: Files over 5MB may take longer to process
  • Date Formats: Use MM/DD/YYYY format for any dates
Tip: If you're unsure about your file format, download our template and copy your data into it.
